# Power BI Superstore Analytics Portfolio Project

## Overview

This project analyzes business sales data using Power BI. Features include:
- Data cleaning and transformation in Power Query
- Data modeling with relationships and dimension tables
- Custom DAX measures for advanced analytics
- Professional dashboard/report design
- Exported PDF for easy viewing

## Files

- Superstore.pbix
- Superstore_Report.pdf

## How to Use

1. Download the .pbix and open in Power BI Desktop.
2. Explore dashboards, model, and metrics.
3. View the PDF for report pages.

## Skills Demonstrated

- Power Query
- DAX measure writing
- Star schema data modeling
- Executive reporting
